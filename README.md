# Dotfiles

Clone repo:
```bash
$ git clone https://github.com/richardkall/dotfiles.git ~/.dotfiles
````

Run script:
```bash
$ cd ~/.dotfiles;
$ ./install
```

Clone Vundle:
```bash
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

Open vim and type `:BundleInstall` to install the bundles.
