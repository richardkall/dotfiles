# Dotfiles

> My personal set of configuration files.

## Setup

Clone repository:

```sh
git clone git@github.com:richardkall/dotfiles.git ~/.dotfiles
```

Bootstrap environment:

```sh
~/.dotfiles/bootstrap
```

## Customize

Add any local customizations to:

- `~/.aliases.local`
- `~/.gitconfig.local`
- `~/.zshrc.local`
