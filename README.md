# Dotfiles

> My personal set of configuration files.

## Setup

```bash
$ git clone https://github.com/richardkall/dotfiles.git ~/.dotfiles
$ sh ~/.dotfiles/bootstrap
```

When adding any new dotfiles, symlink them using:

```bash
$ sh ~/.dotfiles/symlink
```

## Customize

Add any local customizations to:

- `~/.aliases.local`
- `~/.gitconfig.local`
- `~/.vimrc.local`
- `~/.zshrc.local`
